# Web Scraper CSV

This project is a simple Python web scraper that extracts quotes and authors from a website and saves the data to a CSV file.

## What the program does

- sends a request to a webpage
- parses the HTML
- finds quotes and authors
- saves the data into `quotes.csv`

## Technologies used

- `requests`
- `BeautifulSoup`
- `csv`

## Source website

The scraper uses:

https://quotes.toscrape.com

This website is designed for practicing web scraping.
